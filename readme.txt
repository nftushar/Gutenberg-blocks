== Installation ==

1. Upload the `blockxpert` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to BlockXpert settings to configure your blocks
4. Start using the blocks in the Gutenberg editor

== Frequently Asked Questions ==

= Do I need an OpenAI API key? =

Yes, for AI-powered features like FAQ generation and product recommendations, you'll need an OpenAI API key. You can get one from [OpenAI's platform](https://platform.openai.com/).

= Is WooCommerce required? =

WooCommerce is required for product-related blocks (Product Slider and AI Product Recommendations). Other blocks work independently.

= Can I customize the appearance? =

Yes! Each block includes extensive customization options including themes, layouts, and styling controls.

= Are the blocks mobile responsive? =

Absolutely! All blocks are fully responsive and work great on mobile devices.

= Can I disable specific blocks? =

Yes, you can enable/disable individual blocks through the BlockXpert admin settings.

= Is the plugin compatible with my theme? =

BlockXpert is compatible with all modern WordPress themes that support Gutenberg blocks.
